To work with PHP and MySQL using WAMP Server, we need to follow these steps:
1. Start WAMP Server:
Launch WAMP Server. Ensure all services(Apache, MySQL)are running.
2. Create a Database and Tables:
Open phpMyAdmin in web browser (http://localhost/phpmyadmin/).
The default username for phpMyAdmin in WAMP Server is usually root, and by default, there is no password set. When logging in for the first time, you would typically enter root in the username field and leave the password field blank.
Create a new database called myDB.(if it is already exists, ignore).
To store and read images in a database using PHP, we typically use a BLOB (Binary Large Object) field in database table to store the image data.
Create a images table under myDB database. (use 4 columns - id(primary key - make it auto increment), image_name VARCHAR(255), image_type VARCHAR(50), image_data BLOB) and save the table.
Once database (myDB) and table (images) are created, we can create PHP files to interact with the MySQL database. We can use mysqli or PDO for database operations.
